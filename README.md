🎮 Quiz Concurseiro - Missão Aprovação
Um jogo de perguntas e respostas (Quiz) gamificado focado em Concursos e Vestibulares, com design moderno inspirado em Duolingo e jogos de cartas/RPG minimalistas.
 HTML5 


 CSS3 


 JavaScript 


 Tailwind CSS 
 
✨ Funcionalidades

🏠 Tela de Configuração

Escolha do foco de estudo (Vestibular UnB, Concurso TSE, ENEM, PF, PRF)

Seleção da área de conhecimento (Engenharia de Software, Direito Civil, Matemática Financeira, Português, Raciocínio Lógico)

Quantidade de questões: 5, 10 ou 15


Tempo limite por questão: Sem tempo, 30s, 1min ou 2min


🎯 Tela do Jogo

Barra de progresso animada

Indicador de nível do jogador

Timer regressivo com alertas visuais (verde → amarelo → vermelho)

Validação instantânea das respostas

Efeitos visuais: confete para acertos, shake para erros

🏆 Tela de Resultados


Resumo de acertos e erros com gráfico circular animado

Sistema de XP e níveis

Lógica de progressão/regressão de nível

Análise do Oráculo: explicações das questões erradas
